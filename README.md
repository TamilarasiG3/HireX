📌 PROJECT OVERVIEW
HireX

HireX is an AI-powered career intelligence platform that helps students prepare for placements using structured daily practice.

🧠 CORE FEATURES
💻 Daily Coding Practice
📊 Aptitude Tests
🎤 Mock Interviews
🤖 AI Mentor Chatbot
📈 Placement Readiness Score
🧩 Skill Gap Analysis
⚡ DAILY PREPARATION PLAN
Activity	Time	Module
Coding Practice	30 min	DSA
Mock Test	20 min	Aptitude
Mock Interview	10 min	AI Interview
🔐 SECURITY
JWT Authentication
bcrypt Password Hashing
Protected Routes
Input Validation
CORS Enabled
📊 SCORING FORMULA
Placement Readiness Score =
(Coding + Aptitude + Communication) / 3
🚀 DEPLOYMENT
Frontend → Vercel
Backend → Node.js
Database → MongoDB Atlas
🎯 CONCLUSION

HireX is a complete AI-driven placement ecosystem that builds consistency, skill, and confidence for students preparing for tech careers 🚀
